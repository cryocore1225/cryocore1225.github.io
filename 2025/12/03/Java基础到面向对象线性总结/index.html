Java面向对象

Java 基础到面向对象线性总结

1. 内存与存储

Java 程序在运行时主要使用两块内存：

  • 栈（Stack）：
    • 存储基本类型变量（int、double、boolean 等）和引用变量的地址
    • 生命周期随方法调用结束而结束
  • 堆（Heap）：
    • 存储对象实例（Object、数组等）
    • 由垃圾回收器自动管理（Garbage Collector）

基本类型直接存值，引用类型存地址。

2. 数据类型

基本类型（值类型）

类型 说明
byte 8 位整数
short 16 位整数
int 32 位整数
long 64 位整数
float 32 位浮点数
double 64 位浮点数
char 单个字符
boolean true / false

引用类型

  • 类（Class）
  • 接口（Interface）
  • 数组（Array）
  • 枚举（Enum）

引用类型变量存储对象在堆中的地址。

3. 变量与作用域

  • 局部变量：在方法内部声明，栈中存储，方法结束销毁
  • 成员变量（属性）：在类中声明，但方法外
    • 实例变量：属于对象，存堆中
    • 静态变量（static）：属于类，存方法区

变量修饰符

  • final：变量不可修改（基本类型值不可变，引用类型引用不可变）
  • static：属于类，不属于对象
