Math 常用函数

一、取值相关（最常用）

1️⃣ Math.abs(x) —— 绝对值

常见用途：

  • 比较浮点数
  • 计算距离、误差

2️⃣ Math.max(a, b) / Math.min(a, b) —— 最大 / 最小

常见用途：

  • 取上下限
  • 防止越界

二、四舍五入 & 取整（非常常见）

3️⃣ Math.round(x) —— 四舍五入

⚠️ 返回类型是 int / long

4️⃣ Math.floor(x) —— 向下取整

📌 永远向更小的方向取整

5️⃣ Math.ceil(x) —— 向上取整

📌 永远向更大的方向取整
