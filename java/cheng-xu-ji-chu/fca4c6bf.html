9. 小结

  • char 是基本类型，表示一个 Unicode 字符，占 2 字节。
  • String 是引用类型，用于表示字符串。
  • 引用类型变量可以为 null，但空字符串 "" 不是 null。
  • 字符串不可变：字符串内容不可修改，但变量可以改变指向。
  • 可用转义字符表示特殊字符。
  • 字符串支持 + 连接，也可使用 Java 13+ 的文本块。
