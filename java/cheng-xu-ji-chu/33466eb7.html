12. 小结

  • Java 有两类变量：基本类型 与 引用类型
  • 基本类型包括：整数、浮点、布尔、字符
  • 变量可重复赋值，= 是赋值，不是等号
  • 常量（final） 初始化后不可改变
  • var 可自动推断类型
  • 变量作用域受 {} 控制，应尽可能缩小作用域
